# 前端学习路线桌面应用 - 使用指南

## 快速开始

### 首次启动

1. 启动应用后,您将看到欢迎页面
2. 点击"开始学习"按钮,进入学习路线图
3. 选择您想学习的课程节点开始学习

## 功能介绍

### 1. 学习路线图

- **查看学习路径**: 展示完整的前端技术学习路线
- **阶段分级**: 基础入门、进阶提升、高级深化、实际业务场景
- **进度可视化**: 绿色进度条显示各章节完成情况
- **快速跳转**: 点击课程节点直接进入学习

### 2. 课程学习

- **Markdown 内容**: 清晰的知识点讲解
- **代码高亮**: 代码示例支持语法高亮
- **进度标记**: 学习完成后点击"标记为完成"
- **课程导航**: 使用"上一课"/"下一课"按钮切换课程

### 3. 实战练习

- **编辑器练习**: 在线编写 HTML/CSS/JavaScript 代码
- **挑战题**: 通过测试用例验证代码正确性
- **项目实战**: 综合运用知识完成实际项目
- **难度分级**: 简单、中等、困难

### 4. 学习进度

- **统计数据**: 查看已完成课程数、练习数、学习时长等
- **进度可视化**: 各阶段学习进度图表
- **最近学习**: 显示最近访问的课程
- **成就系统**: 解锁学习成就徽章

### 5. 个人中心

#### 外观设置
- **深色模式**: 切换浅色/深色主题,保护眼睛
- **字体大小**: 调整阅读字体大小(14px-20px)

#### 数据管理
- **导出数据**: 将学习进度保存为 JSON 文件
- **导入数据**: 从备份文件恢复学习进度
- **清空进度**: 重置所有学习记录(谨慎使用)

## 学习路线说明

### 基础入门阶段

适合零基础学习者,包含:
- HTML 基础知识和语义化标签
- CSS 样式、布局和响应式设计
- JavaScript 基础语法和 DOM 操作

**建议学习时长**: 6-8 周

### 进阶提升阶段

适合有一定基础的开发者,包含:
- ES6+ 现代 JavaScript 特性
- 前端工程化工具(pnpm, Vite, Git)
- React 框架基础

**建议学习时长**: 8-10 周

### 高级深化阶段

适合进阶开发者,包含:
- React Hooks 深入应用
- 状态管理方案(Redux, Zustand)
- TypeScript + React
- 性能优化技巧

**建议学习时长**: 8-12 周

### 实际业务场景阶段

适合高级工程师,包含:
- Canvas 数据可视化
- 微前端架构
- 复杂表单和文件处理
- 跨端通信和权限管理

**建议学习时长**: 持续学习

## 学习建议

### 1. 制定学习计划

- 每天保持 1-2 小时的学习时间
- 完成每个章节的练习题
- 定期复习已学内容

### 2. 理论与实践结合

- 学完知识点后立即进行练习
- 尝试自己编写代码实现示例
- 遇到问题先思考,再查看答案

### 3. 记录学习笔记

- 在课程页面记录重点和疑问
- 整理知识点形成个人笔记
- 定期回顾和总结

### 4. 项目实战

- 完成各阶段的项目实战
- 尝试独立开发小项目
- 将所学知识应用到实际场景

## 快捷键

- **Ctrl/Cmd + S**: 保存进度(自动保存)
- **Ctrl/Cmd + E**: 导出数据
- **Ctrl/Cmd + I**: 导入数据

## 常见问题

### Q: 如何备份学习进度?

A: 进入"个人中心" → "数据管理" → 点击"导出学习数据",选择保存位置即可。

### Q: 更换电脑后如何恢复进度?

A: 在新电脑上安装应用后,进入"个人中心" → "数据管理" → 点击"导入学习数据",选择之前导出的文件。

### Q: 课程内容可以离线访问吗?

A: 是的,所有学习内容都内置在应用中,无需联网即可学习。

### Q: 如何重新学习某个章节?

A: 直接从学习路线图中点击相应的课程节点即可重新学习,之前的进度不会丢失。

### Q: 练习题的参考答案在哪里?

A: 完成练习后,点击"查看答案"按钮即可查看参考实现。
